* Roles and Authentication - YES
* Relational Database- YES
* Bonus- Delete Roles- YES
* Role based Links Hide- YES
* Role based controller Access- YES
* CSS Slate Template- YES
* Drowpdown for Register/ Logout- YES
* Carousel- YES
